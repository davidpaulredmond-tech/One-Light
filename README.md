# One Light

This is a simple static PWA version of One Light.

## Why this version is easier
This package uses plain HTML, CSS, and JavaScript, so Vercel can deploy it as a static site without a React build step.

## Upload to GitHub
Upload all files in this folder to the root of your GitHub repository.

## Deploy to Vercel
1. Sign in to Vercel with GitHub.
2. Import your repository.
3. Keep the default settings.
4. Click Deploy.

Because this is a static site, there is no npm build dependency required for Vercel to publish it.
